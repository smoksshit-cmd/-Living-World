# 🌍 Living World — SillyTavern Extension

Расширение для SillyTavern, которое возвращает **живость миру** в ролевых играх.

## Что делает

- **🎲 Случайные энкаунтеры** — НПС появляются рандомно каждые N сообщений с настраиваемым шансом
- **👤 Умные имена** — авто-определяет стиль имён по контексту (русские/иностранные), или выбирается вручную
- **🧠 Автономия персонажа** — основной персонаж живёт своей жизнью, пока ты ведёшь ветки с НПС
- **🔒 Разделение знаний** — персонаж не знает о сценах, при которых он не присутствовал
- **📜 Хроника мира** — события инжектятся в промпт как память мира
- **📖 Lorebook интеграция** — НПС автоматически сохраняются в World Info
- **🔌 Отдельный AI эндпоинт** — НПС генерируются через отдельный API, не тратя токены основного чата

---

## Установка

### Через SillyTavern (рекомендуется)

1. В SillyTavern: `Extensions` → `Install Extension`
2. Вставь URL: `https://github.com/ВАШ_ЮЗЕР/living-world-extension`
3. Нажми Install

### Вручную

```bash
cd SillyTavern/public/scripts/extensions/third-party
git clone https://github.com/ВАШ_ЮЗЕР/living-world-extension living-world
```

Перезапусти SillyTavern.

## Настройки

### Стиль имён НПС

| Вариант | Описание |
|---------|----------|
| 🔍 Авто | Анализирует последние 20 сообщений и определяет стиль |
| 🇷🇺 Русские | Алексей, Дмитрий, Анна, Полина... |
| 🌐 Иностранные | Alex, Ethan, Chloe, Luna... |
| 🎲 Смешанные | Случайно из обоих пулов |

Пол: Любой / Только мужские / Только женские

---

### Случайные энкаунтеры

- **Каждые N сообщений** — минимальный интервал между бросками кубика
- **Шанс появления** — % вероятность при каждом броске (1–100%)
- **Веса типов:**
  - 👤 **Проходной** — живёт мир, уходит через сцену
  - 🔍 **С зацепкой** — несёт информацию, конфликт, квест
  - ⭐ **Важный** — потенциально долгосрочный персонаж с тайной

---

### Отдельный AI эндпоинт

Подключи любой OpenAI-совместимый API:
- OpenAI
- OpenRouter
- LM Studio (локально)
- Ollama
- Kobold / Oobabooga с OpenAI-совместимым режимом

**Поля:**
- URL: `https://api.openai.com/v1` или локальный `http://localhost:1234/v1`
- API ключ
- Модель (загружается кнопкой «Загрузить»)
- Кнопка «Тест соединения» — проверяет что всё работает
